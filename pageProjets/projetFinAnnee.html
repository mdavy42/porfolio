Projet de fin de 1ere année de BUT

Ce projet est à la destination d'un client fictif.
Une partie de ce projet était consacrée au management de projet,
avec notament la réalisation d'un Gantt

Le projet melange plusieurs langages de programation :

  • C#
  • Python
  • SQL

Ce projet de fin d'anne m'a permis d'utiliser toutes mes competences apprisent durant l'annee en un seul projet. Pour mener le projet à bien, l'application doit être capable de visualiser un fichier CSV et pour cela, nous avons décidé de créer une base de données à partir des fichiers CSV. Cette base de données nous sert d'abord à stocker les fichiers CSV et sera alors reliée à un script Python qui permet de réaliser les graphiques attendus par l'utilisateur. Ce même script Python sera relié à notre interface I.H.M. en C# qui permettra à notre client de choisir les données qu'il souhaite afficher. Pour choisir ces données, il pourra les rechercher par des tags correspondant à un thème, un pays, une organisation… Puis ces mêmes données seront affichées sous le format désiré (Histogramme, boîte à moustache etc...). L'utilisateur pourra aussi insérer de nouveaux jeux de données dans la base de données, mais cela pourrait ne pas se faire directement sur le logiciel, on pourra au minimum en insérer depuis le script SQL, un fichier sera fourni à cet effet. La solution doit être déployable sur les machines de l'entreprise, sur des exploitations Windows. Le logiciel devra être intuitif et ergonomique pour l'utilisateur. Il n'est pas nécessaire d'avoir une connexion internet pour utiliser le logiciel.
